# Домашнее задание: Работа с файловой системой

## Описание
Скрипт для объединения лога визитов с данными о покупках.

## Результаты обработки
- Всего визитов: 199,999
- Визиты с покупками: 13,798
- Уникальных покупателей: 13,533
- Конверсия: 6.90%

## Файлы в репозитории
- `hw_filesystem_v.5.py` - основной скрипт
- `funnel.csv` - результат (визиты с категориями покупок)
- `reports/statistics_*.json` - детальная статистика

## Подготовка данных

1. Скачать архив с домашним заданием (ссылка в материалах к лекции)
2. Распаковать архив
3. Скопировать папку `Downloads` в `python_path/`
4. Скопировать папку `data` в `python_path/`

## Запуск

```bash
cd python_path
python hw_filesystem_v.5.py

Скрипт автоматически ищет файлы в папках:
- Downloads/
- data/
- в папке со скриптом

Важные особенности
✅ Используется модуль csv для корректной обработки CSV
✅ Поддержка UTF-8 (файлы открываются в Excel без кракозябр)
✅ Построчная обработка больших файлов (не загружает всё в память)
✅ Сохранение статистики в JSON
✅ Автоматический поиск файлов с данными
✅ Подробный вывод прогресса обработки

Важно:
Если при запуске возникает ошибка "НЕ НАЙДЕНЫ ФАЙЛЫ С ДАННЫМИ":
Убедитесь, что файлы purchase_log.txt и visit_log.csv скачаны из архива к ДЗ
Поместите их в папку Downloads/ внутри python_path/
Запустите скрипт снова
Команда для запуска:
python hw_filesystem_v.5.py
